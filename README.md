# Internet Explorer Reborn 3.0

Internet Explorer Reborn 3.0 is the current standalone Windows build of the project, combining a classic Internet Explorer-inspired shell with optional Chromium-backed browsing.

## Included In 3.0

- `build\Internet Explorer Reborn 3.0.exe`
- `build\Microsoft.Web.WebView2.Core.dll`
- `build\Microsoft.Web.WebView2.WinForms.dll`
- `build\WebView2Loader.dll`
- `build\app.ico`

## Core Features

- Classic Internet Explorer-style shell
- `Modern Mode` using WebView2
- Multi-tab browsing with close buttons and `+` new tab support
- Windows 8.1-style Home menu
- Favorites, Start Favorites, and browsing history
- Search provider selection
- Separate Downloads window
- Custom icon and release-ready build output

## Main Source Files

- `Program.cs`
- `MainForm.cs`
- `Dialogs.cs`
- `AppSettings.cs`
- `build.ps1`
- `app.manifest`

## Running The App

1. Open the `build` folder.
2. Launch `Internet Explorer Reborn 3.0.exe`.

## Notes

- Classic mode is built for the nostalgic experience first.
- Modern websites work best with `Modern Mode` enabled.
- This project is fan-made and not affiliated with Microsoft.
